## ğŸ“– About the Project
**Boba** is a cross-platform mental health tracking system designed to bridge the "data blackout" between therapy sessions. 

For **Patients**, it offers a low-friction iOS app to log daily moods, activities, and journal entries in seconds.  
For **Therapists**, it provides a web-based data dashboard to visualize patient progress and identify trends over time.

By synchronizing emotional data in real-time, Boba transforms therapy from a reactive weekly conversation into a proactive, data-informed journey.
